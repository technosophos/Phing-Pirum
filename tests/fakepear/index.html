Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover fakepear.alpeh-null.tv
Listing available packages: 
pear remote-list -c fakepear
Installing a package: 
pear install fakepear/package_name
Installing a specific version/stability: 
pear install fakepear/package_name-1.0.0
pear install fakepear/package_name-beta
Receiving updates via a feed: 
http://fakepear.alpeh-null.tv/feed.xml
